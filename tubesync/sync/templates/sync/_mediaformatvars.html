Available media name variables

Name Description Output example
{yyyymmdd} Media publish date in YYYYMMDD 20210131
{yyyy_mm_dd} Media publish date in YYYY-MM-DD 2021-01-31
{yyyy} Media publish year in YYYY 2021
{mm} Media publish month in MM 01
{dd} Media publish day in DD 31
{source} Lower case source name, max 80 chars my-source
{source_full} Full source name My Source
{uploader} Uploader name Some Channel Name
{title} Lower case media title, max 80 chars my-video
{title_full} Full media title My Video
{key} Media unique key or video ID SoMeUnIqUeId
{format} Media format string 720p-avc1-mp4a
{playlist_title} Playlist title of media, if it's in a playlist Some Playlist
{video_order} Episode order in playlist, if in playlist (can cause issues if playlist is changed after adding) 01
{ext} File extension mkv
{resolution} Resolution 720p
{height} Media height in pixels 720
{width} Media width in pixels 1280
{vcodec} Media video codec avc1
{acodec} Media audio codec opus
{fps} Media fps 60fps
{flag_hdr} Media has HDR flag hdr
